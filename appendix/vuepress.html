# Contribute to this tutorial

This tutorial is created using the VuePress static site generator. To contribute to the tutorial please let the site manager add you as a collaborator. Then, you can directly edit using the "Edit this page" link on each page. After the commit, the pages will be automatically re-built through Travis-CI (expecting a few minutes' delay).

Alternatively, you can write and build the tutorial on your own computer. You need to first install node.js and npm (e.g. on ubuntu). I recommend using nvm for better version control.

# Install dependencies


 
 
 

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash # or newer versions
source ~/.bashrc
nvm install node
npm install
